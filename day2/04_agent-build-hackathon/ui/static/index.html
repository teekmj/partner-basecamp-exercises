Drafting (with tool use) + Review (no tools)

        

      


      
Pipeline stages

      

        

          
1
Parse

          
no LLM · pure Python

          
JSON / numbered / bulleted / paragraph input → list of {id, category, text}

        

        

          
2
Retrieve

          
no LLM · keyword scoring

          
Token-overlap + category boost (+5) → top-3 KB entries

        

        

          
3
Draft

          
LLM + tool loop (≤5 turns)

          
Agent calls search_kb, synthesizes JSON answer with sources + confidence + flags

        

        

          
4
Review

          
LLM, no tools

          
Holistic pass · returns {consistency_score, issues, recommendations}

        

        

          
5
Export

          
no LLM · JSON + HTML

          
Final deliverable as JSON or printable HTML (browser → PDF)

 ┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐     ┌────────┐

                                                                         │

                                                              │  HTML / PDF / save  │

Failure modes

      

        
FailureMitigation

        
          
LLM returns malformed JSON_extract_json tries fenced + raw + embedded; fallback shape on failure

          
Tool loop never endsmax_turns=5 · returns flags: [max_turns_exceeded]

          
Reviewer truncates JSONmax_tokens=4096 + robust extractor

          
Reviewer false positivesSharpened prompt distinguishes contradiction from coverage gap

          
API key missingBackend HTTP 400 · UI inline error

          
Server thread crashProducer thread try/except emits {type: error} over SSE
